# Bank-Reconciliation-Project
BRS project using Tally with real-world scenarios
##  Objective
To reconcile bank transactions recorded in company books with the bank statement.

---

## Process

### 1. Voucher Entries
Transactions were recorded in Tally including receipts and payments.

### 2. Bank Ledger
Bank ledger was prepared to track all bank-related transactions.

### 3. Bank Statement
Bank statement was created in Excel.

### 4. BRS Working
Unreconciled transactions were identified.

### 5. BRS Summary
Final reconciliation was completed.

---

##  Key Findings

- ₹5000 (Rahul Traders) not reflected in bank
- ₹3500 (Rahul Traders) not reflected in bank

 Total Difference: ₹8500

---

##  Tools Used
- Tally Prime
- Microsoft Excel
